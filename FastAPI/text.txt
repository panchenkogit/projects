Выполненные задачи:
Настройка структуры проекта: Организовали папки frontend/page для разметки и frontend/static для стилей и скриптов, закрепив эту структуру в FastAPI приложении для корректной работы со статикой.

Создание интерфейса добавления продуктов:

Верстка для формы, где пользователи могут вводить название продукта, описание и показатели белков, жиров, углеводов и калорий.
Установка стандартных значений для полей, валидация данных, и адаптация под удобный интерфейс 2024 года.
Стилизация формы и страниц:

Цветовая схема выбрана в голубых тонах с акцентом на приятные, современные оттенки.
Элементы формы имеют закруглённые края и удобное расположение для лучшей читаемости и простоты использования.
Добавили элементы всплывающего уведомления, которое сообщает об успешном добавлении продукта.
Подключение и обработка скриптов:

Создали script.js для отправки данных формы на сервер по API /product/add.
Реализовали плавное всплывающее окно уведомления об успешном добавлении продукта.
Отображение таблицы продуктов:

Настроили таблицу для отображения списка продуктов в формате скроллинга, закрепив её справа от формы добавления.
Настроили автоматическую загрузку данных из /products, где каждый раз отображается до 100 продуктов, а при большем количестве данные подгружаются по мере необходимости.
